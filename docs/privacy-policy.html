Privacy Policy

Last updated: May 5, 2025

ReelRanker is a Chrome extension that scrapes Instagram reel comments, ranks them using AI, and lets you build and export tier lists. This policy explains what data we access and how it is handled.

Data We Access

When you use ReelRanker on an Instagram reel page, the extension reads:

  • The reel caption and username visible on the page
  • Comments and like counts visible in the DOM
  • The profile picture URL of the reel author

This data is scraped directly from the page you are viewing. It is never sent to any ReelRanker server — there is no backend.

API Keys

ReelRanker requires you to provide your own API keys for Gemini (Google) and OpenAI. These keys are:

  • Stored locally in Chrome's chrome.storage.local on your device only
  • Never transmitted to any ReelRanker server
  • Sent directly from your browser to Google and OpenAI when you trigger a ranking or TTS action

Your use of those APIs is governed by Google's Privacy Policy and OpenAI's Privacy Policy.

Session Data

Tier list sessions (reel info, ranked comments, ranking mode) are saved locally in your browser using IndexedDB. This data never leaves your device and can be deleted at any time by clearing the extension's storage in Chrome settings.

Data Sharing

We do not collect, store, sell, or share any personal data. ReelRanker has no analytics, no telemetry, and no backend servers.

Third-Party Services

When you use AI features, scraped reel and comment text is sent to the respective API provider (Google Gemini or OpenAI) using your own API key. Please review their privacy policies before use.

Permissions Explained

  • sidePanel — displays the extension UI alongside Instagram
  • activeTab — reads the current tab to access the reel page
  • scripting — injects the comment scraper into Instagram pages
  • storage — saves API keys and sessions locally on your device

Children's Privacy

ReelRanker is not directed at children under 13. We do not knowingly collect any information from children.

Changes to This Policy
